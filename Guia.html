Promesas

Es un objeto que permite procesar el resultado, terminación o fracaso de una petición asíncrona Ajax que tarda en dar respuesta, Nos retorna un valor o si falla un error, permite procesarlo de forma asincrona cuando llegue ese valor.

  • resolve: resultado
  • reject: error, no hay resultado
  • setTimeout: Tiempo de espera milisegundos a que algo se ejecute

Declaracion de la promesa


var saludar = new Promise((resolve, reject) => {

    setTimeout(() => {    //Tiempo de ejecución 

        let saludo = "Hola Mundo";

        if(saludo){
            resolve(saludo);
        }else{
            reject('no hay saludo disponible')
        }
    },2000)  
})

Procesando el resultado


saludar.then(resultado =>{   //.then Procesa el resultado
    alert(resultado);
})
.catch(error =>{   //.catch opcional captura el posible error
    alert(error);
});

Clases en POO

Es basicamente una plantilla que sirve para crear o definir varios objetos, con las caracteristicas definidas en la misma, estan compuestas por metodos y su método principal siempre es el constructor

  • constructor: metodo inicial de la clase, donde se definen los atributos o caracteristicas, puede recibir parametros
  • Métodos generales: Es donde se definen las funciones o acciones que puede hacer la clase
  • Parametros: Valores que recibe un método

Creando la clase


class Coche{

    constructor(marca,velocidad,modelo){
        this.marca = marca,
        this.velocidad= velocidad,
        this.modelo = modelo;
    }
    aumentarVelocidad(){
        this.velocidad += 1;
    }
    reducirVelocidad(){
        this.velocidad -=1;    }
}

// Definiendo el objeto con base a la clase
