Javascript

Como iniciar

Para trabajar con JS debemos tener un HTML en el cual cargarlo, y acontinuacion tenemos 2 alternativas para integrarlo al HTML,
1. Escribir codigo dentro de la etiqueta script de html (No se recomienda)
2. Enlazar un archivo JS externo con el atributo src de la etiqueta script indicando la ruta relativa del archivo js, es la forma mas recomendada y optima lo ideal es cargarlo al final del archivo, por el orden de lectura del HTML y Javascript.
