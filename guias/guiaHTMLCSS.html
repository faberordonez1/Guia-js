Responsive Web Design

Técnicas para adaptar un sitio web a los diferentes tipos de pantallas, ya que existen diferentes densidad de pixelees y tamaños de pantalla

Patrones Responsive

  1. Mostly Fluid
  2. Colocacion de Columnas
  3. Layaout Shifter
  4. Tiny Tweks
  5. Off Canvas (Menú de hamburguesa)

Para consultar sobre patrones de diseño ingresar a mediaqueri.es

Portrait - Landscape

Este concepto se utiliza para definir el modo de pantalla, generalmente se refiere a como se esta utilizando la pantalla, en forma vertical(Portrait) u Horizontal (Landscape)

Mobil First o Desktop First

Es el modo como se abordara el resposive design, empezar de menor a mayor (Mobile first) tamaño de pantalla o mayor a menor (Desktop first)

Developers Tools F12

Estas herramientas vienen integradas en los navegadores para facilitar el desarrollo web, entre ellas tenemos la siguientes

  • Desplegable Superior Permite cambiar entre tamaños de pantalla ya definidos o de forma libre (resize)
  • Zoom SuperiorPermite alejar o acercar el tamaño de pantallas
  • Boton rotacionEste boton permite cambiar entre portrait y Landscape
  • 3 Puntos superiores Esta opcion nos permite acceder a las reglas, tambien nos permite mostrar los mediaqueris del sitio (Azul Max width, Rangos Verde, Naranja Min-width)y la seccion de capturas de pantalla entre otras

Metaviewport

Viewport Es el area visible del navegador

Metaviewport es la etiqueta de metadatos que permite adaptar el contendio de forma adecuada, segun el tipo de pantalla, ya que sin esta etiqueta, el contenido se reescala para intentar mostrarse como si fuera PC y el website posiblemente no sea legible


<meta name="viewport" content="width=device-width, initial-scale=1.0">
