Responsive Web Design

Técnicas para adaptar un sitio web a los diferentes tipos de pantallas, ya que existen diferentes densidad de pixelees y tamaños de pantalla

Patrones Responsive

  1. Mostly Fluid
  2. Colocacion de Columnas
  3. Layaout Shifter
  4. Tiny Tweks
  5. Off Canvas (Menú de hamburguesa)

Para consultar sobre patrones de diseño ingresar a mediaqueri.es

Portrait - Landscape

Este concepto se utiliza para definir el modo de pantalla, generalmente se refiere a como se esta utilizando la pantalla, en forma vertical(Portrait) u Horizontal (Landscape)

Mobil First o Desktop First

Es el modo como se abordara el resposive design, empezar de menor a mayor (Mobile first) tamaño de pantalla o mayor a menor (Desktop first)

Developers Tools F12

Estas herramientas vienen integradas en los navegadores para facilitar el desarrollo web, entre ellas tenemos la siguientes

  • Desplegable Superior Permite cambiar entre tamaños de pantalla ya definidos o de forma libre (resize)
  • Zoom SuperiorPermite alejar o acercar el tamaño de pantallas
  • Boton rotacionEste boton permite cambiar entre portrait y Landscape
  • 3 Puntos superiores Esta opcion nos permite acceder a las reglas, tambien nos permite mostrar los mediaqueris del sitio (Azul Max width, Rangos Verde, Naranja Min-width)y la seccion de capturas de pantalla entre otras

Metaviewport

Viewport Es el area visible del navegador

Metaviewport es la etiqueta de metadatos que permite adaptar el contendio de forma adecuada, segun el tipo de pantalla, ya que sin esta etiqueta, el contenido se reescala para intentar mostrarse como si fuera PC y el website posiblemente no sea legible


<meta name="viewport" content="width=device-width, initial-scale=1.0">
Width=> Ancho  initiaL-scale =>Zoom Rango (0 a 1)

Unidades Relativas (%, em, rem)

Porcentaje: Unidad que toma tamaño de acuerdo a su elemento padre o contendedor

Viewport (vw,vh) Unidad relativa con respecto al tamaño del viewport (pantalla visible), puede tomar valores de 0 a 100, cada tamaño indica el porcentaje segun la pantalla

REM Unidad relativa referente al ancentro mas lejano, puede ser Html o en su defecto body, el body por defecto ya tiene asignado un tamaño de fuente en comparación al html

EM Unidad relativa referente al contenedor mas cercano o a el mismo, con tamaño de fuente asignada, esa fuente especificada es igual a 1em, a continuacion podemos ver 1 ejemplo


<nav> font-size:16px;
    <ul> font-size:2em; 32px
        <li> font-size:1em; 32px
        <a href=""> </a> (font-size:.5em; 16px) (padding:2em; 32px)
            //padding toma el tamaño de acuerdo al font size
        </li>
    </ul>
</nav>

Mediaqueris

Son el eje principal del responsive design, anteriormente se utilizaba un subdominio m.dominio y era un website totalmente diferentes para moviles

Los mediaqueris se componen de un media type y una o mas condiciones, y tiene la siguiente estructura


//Estructura gral
@media media type and (condición){
    code CSS
}

//Pantallas con ancho inferior o igual a 768
@media screen and (max-width:768px){
    code CSS
}
//Pantallas con ancho de 480 hasta 768
@media screen and (max-width:768px) and (min-width:480){
    code CSS
}

Los mediaqueris se pueden utilizar de acuerdo al modelo o necesidad con las siguientes condiciones, mobile first desde pequeno a grande o desktop first de grande hasta pequeño,y las condiciones las podemos relacionar como (min-width = desde) y (max width= hasta)

Mobile first

@media screen and (min-width:320px){ }
@media screen and (min-width:480px){ }
@media screen and (min-width:768px){ }
@media screen and (min-width:1024px){ }
Desktop first

@media screen and (max-width:320px){ }
@media screen and (max-width:480px){ }
@media screen and (max-width:768px){ }
@media screen and (max-width:1024px){ }
Formas de insertar mediaqueris

Existen 3 formas de agregar mediaqueris a un proyecto. Enlazando una hoja de estilos condicionada en el head del Html, la segunda opcion es aplicandola en la hoja de estilos ya enlazada y la tercera forma es utlilizando una estiqueta style directamente en el Html


1. <link rel="stylesheet" href="css/media.css" media=" screen and (max-width:768px)">

2. @media screen and (max-width:768px){ }

3.<style>
    @media screen and (max-width:1024px){ }
</style>
